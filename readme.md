# Curso Maquetacion CSS

Colocando los siguientes estilos en la etiqueta `body` logramos que no sea scrolleable la pagina
```CSS
body {
    position: fixed;
    top: 0;
    bottom: 0;
    left: 0;
    right: 0;
}
```
